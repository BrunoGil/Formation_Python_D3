# The data

`clean.csv` is the **cleaned Superstore data from Day 2** — already bundled, so
Day 3 starts where Day 2 left off (no need to have finished Day 2).

Columns include `order_date`, `region`, `category`, `segment`, `sales`,
`quantity`, `discount`, `profit`. It originally comes from the
[Sample - Superstore](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
dataset on Kaggle.
