# Python for Data Analytics

Training materials for a 4-day (3h/day) program introducing Python for data
analytics.

## Program

| Day | Topic |
|-----|-------|
| [Day 1](day-01/) | Python Foundations + Getting Your Data In |
| [Day 2](day-02/) | Manipulating & Cleaning Data |
| [Day 3](day-03/) | Visualizing & Reporting |
| Day 4 | From Notebook to Pipeline (+ Airflow awareness) |

Each day has its own folder with a `README`, teaching notebook, exercises, and
solutions. Day 1 runs in **Google Colab**; Day 2 runs in **GitHub Codespaces**
from a template repo. See each day's README for how to launch — start with
[`day-01/README.md`](day-01/README.md).
